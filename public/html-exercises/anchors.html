Anchors

Hyperlinks connect Web pages, or data items, to one another. In HTML, anchor elements define the hyperlinks you see when you browse the Web. An anchor can create a link from a part of a Web page, such as a text string or image, to another site, page or even a particular point within a page.

Attributes

The most important attribute of an anchor element is the href.
Attribute Meaning
href means Hypertext REFerence which specifies the destination address

Local Resource

When you browse the Web, you use hyperlinks to click from one site to another, between pages in the same site and between parts of the same page. To create a hyperlink in HTML, you use an anchor <a> element:

Example

Another Page 
<a href="another.html"> Another Page </a>
The value of the href can be an address for a website, the location of a particular page or even a section within a page.

In the above example, the href indicates a relative URI. This means that the destination location is being described relative to the location of the hyperlink itself. For example, you could use the above href if the another.html page is located in the same directory as the page containing the anchor.

External Resource

To include a link to another site, the full address is specified using a URL.

Example

Google 
<a href="http://www.google.com"> Google </a>

Page Section

Hyperlinks also connect to particular points within pages. This allows anchors to link between the different parts of a single page.

Example

Local Resource 
<a href="#local"> Local Resource </a>
By clicking the link, you will jump from your current part of the page to the "Local Resource" section. To do this, the value of the href should be connected to the id of the section of the page where you want to jump into, in this example, the Local Resource heading. Together with the above code, the Local Resource heading on top of this page should be coded this way: 

<h4 id="local">Local Resource</h4>
The id of the h4 tag allowed the anchor to identify which section of the page to jump into. The href should contain the id of that section preceded by a #, ("#local").

Image Links

So far the hyperlinks we have seen use text as the content of the anchor element (e.g. "another"). The content of an anchor is whatever the user clicks on to visit the place indicated by the href attribute. This clickable object can be another media type or Web page item, but most commonly will be either text or an image:

Example

<a href="http://www.famigo.com/blog/2013/08/15-silly-baby-memes/"><img src="/images/babymeme.jpg" width="450"></a>
Try clicking on the image. You'll be brought to a page at famigo.com. You'll learn more about inserting images in a web page in our future meetings.

Absolute and Relative Paths

The value of the href attribute is not just a simple file name or a location on the web (URL). These values are in general file paths. There are two ways to specify a path: absolute or relative.
Absolute Paths
An absolute path gives the location of a file or directory in reference to a root directory. Absolute paths are easier to use and understand. However, it is not good practice on your own website. If you have to move your web site, you would have to change the paths in order to get it to work.

Example

Another Page 
<a href="C://Users/Miyah/Desktop/another.html"> Another Page </a>
<!--OR-->
<a href="/etc/home/Desktop/another.html"> Another Page </a>
<!--OR-->
<a href="http://example.com/another.html"> Another Page </a>
Relative Paths
An absolute path gives the location of a file or directory in reference to the current working directory. The current working directory denotes the path where your web page is located.

Example

Another Page 
<!--If another page is located in the same directory as the current web page-->
<a href="another.html"> Another Page </a>
<!--If another page is located in a folder located in the directory of the current web page-->
<a href="folder/another.html"> Another Page </a>
<!--If another page is located up one level to the directory of the current web page-->
<a href="../another.html"> Another Page </a>
The hyperlink is one of the fundamental building blocks of the Web, since it creates connections between sites, pages and specific data items. The ability of users to click between the chunks of information on the Web makes the vast amount of data available easier to navigate.
