Images 1

In HTML, images are defined with the <img> tag. The <img> tag is empty, it only contains attributes, and does not have a closing tag. However, a slash can be added before the greater-than sign for XHTML validity.

Attributes

There are two important attributes of an <img> tag:
Attribute Meaning
src stands for "source" which provides the web browser the source of the image
alt stands for "alternative" which provides a text-based alternative for users in case the image will not load, or if they are visually impaired

Example

<img src="baby.jpg" alt="This is a cute baby."/>

Other attributes of the <img> tag are the width, height and align. You can also use the style attribute to specify the width and height of an image.

Example

<img src="baby.jpg" alt="This is a cute baby." width="200px" height="150px"/>

Absolute and Relative Paths

An image, inserted in a web page, can be found locally in your PCs or from the web. It is important that the path of the image is correct, otherwise the alternative text will be displayed. There are two ways to specify a path: absolute or relative.
Absolute Paths
An absolute path gives the location of a file or directory in reference to a root directory. Absolute paths are easier to use and understand. However, it is not good practice on your own website. If you have to move your web site, you would have to change the paths in order to get it to work.

Example

<img src="C://Users/Betel/baby.jpg" alt="This is a cute baby."/>
<!--OR-->
<img src="/etc/home/Downloads/baby.jpg" alt="This is a cute baby."/>
<!--OR-->
<img src="http://example.com/uploads/baby.jpg" alt="This is a cute baby."/>
Relative Paths
An absolute path gives the location of a file or directory in reference to the current working directory. The current working directory denotes the path where your web page is located.

Example

<!--If image is located in the same directory as the web page-->
<img src="baby.jpg" alt="This is a cute baby."/>
<!--If image is located in a folder located in the directory of the web page-->
<img src="folder/baby.jpg" alt="This is a cute baby."/>
<!--If image is located up one level to the directory of the web page-->
<img src="../baby.jpg" alt="This is a cute baby."/>
