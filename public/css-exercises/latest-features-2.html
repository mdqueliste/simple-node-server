Latest Features Exercise

For this exercise, create an html file in the public/css-exercises folder named latest-features-2.html. Save your external stylesheet (css file) in public/css folder as latest-features-2.css. With the given HTML content, design the web page using an external style sheet in such a way it will look exactly like the screenshot below. Make use of what you've learned about from the previous tutorial to be able to correctly design the web page.
Note: You cannot modify the html, only the style sheet! EXCEPT for adding a <link> tag for the Google Font.
In this exercise, you will just improve your box-model-3.html. You'll just make use the properties discussed in latest-features-1.html. Details:
  • All text in the page should have a Google font "Comfortaa".
  • The class "rect" should have a rounded border of 10px.
  • Change the background of the class "whole" into a gradient fading from #84befe to white to the top.
  • The small white box should have a shadow of gray, horizontally and vertically offset by 5px and has a blur of 10px;
  • The text inside the class "preview" should have a font-size of 90px and shadow of gray, horizontally and vertically offset by 5px and has a blur of 10px.
  • The navigation tabs should have a rounded upper right corner of 15px.
  • The left-head and right-head should have a font-size of 30px, rounded bottom corners of 20px and gradient background fading from #84befe to lightblue towards the top.
  • Instead of color blue, the footer text should now be colored white.

HTML

<!DOCTYPE html>
<html>
	<head>
		<link rel="stylesheet" type="text/css" href="latest-features-2.css" />
		<!--Insert your <link> tag for the Google font HERE-->
	</head>

	<body>
		<!--The fixed rectangular box on the right side-->
		<div class="rect"> 
			<a href="https://www.facebook.com/wordpress">Like us on FB</a>
		</div>
		
		<!--The light blue-colored container of everything-->
		<div class="whole">
		
			<!--The invisible container of the small white box and big blue box-->
			<div class="header">
			
				<!--The small white box-->
				<div class="box"></div>
				
				<!--The big blue box-->
				<div class="preview">Theme Preview</div>
			</div>
			
			<!--The container of the middle content-->
			<div class="container">
			
				<!--The container of the navigation Tabs-->
				<div class="nav">
					<a href="#">Home</a>
					<a href="#">About</a>
					<a href="#">Contact</a>
					<a href="#">Feedback</a>
					<a href="#">Donate</a>
				</div>
				
				<!--The white empty space in the middle-->
				<div class="mid"></div>
				
				<!--The container of the last section-->
				<div class="last">
				
					<!--The left box-->
					<div class="left">
					
						<!--The header of the left box-->
						<div class="left-head">Links</div>
					</div>
					
					<!--The right box-->
					<div class="right">
					
						<!--The header of the right box-->
						<div class="right-head">Register</div>
					</div>
				</div>		
			</div>
			
			<!--The footer text-->
			<div class="footer">CMSC2 Midyear 2015</div>
		</div>
	</body>

</html>
