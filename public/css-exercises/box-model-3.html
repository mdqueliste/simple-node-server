Box Model Exercise

For this exercise, create an html file in the public/css-exercises folder named box-model-3.html. Save your external stylesheet (css file) in public/css folder as box-model-3.css. With the given HTML content, design the web page using an external style sheet in such a way it will look exactly like the screenshot below. Make use of what you've learned about the Box Model to be able to correctly design the web page.
Note: You cannot modify the content, only the style sheet!
Details:
  • The font family of the text in the web page should be Arial.
  • The width of the whole container is 900px (excluding the padding).
  • The two shades of blue to be used are: #84befe (darker) and #cbe5ff (lighter)
  • Each navigation tab changes color and background color when mouse hovers over it.
  • The Like link in a rectangular box is fixed with top:200px and right:-5px.
  • The rest of the sizes are up to your judgement to make it look like the screenshots below.

HTML

<!DOCTYPE html>
<html>
	<head>
		<link rel="stylesheet" type="text/css" href="../css/box-model-3.css" />
	</head>

	<body>
		<!--The fixed rectangular box on the right side-->
		<div class="rect"> 
			<a href="https://www.facebook.com/wordpress">Like us on FB</a>
		</div>
		
		<!--The light blue-colored container of everything-->
		<div class="whole">
		
			<!--The invisible container of the small white box and big blue box-->
			<div class="header">
			
				<!--The small white box-->
				<div class="box"></div>
				
				<!--The big blue box-->
				<div class="preview"></div>
			</div>
			
			<!--The container of the middle content-->
			<div class="container">
			
				<!--The container of the navigation Tabs-->
				<div class="nav">
					<a href="#">Home</a>
					<a href="#">About</a>
					<a href="#">Contact</a>
					<a href="#">Feedback</a>
					<a href="#">Donate</a>
				</div>
				
				<!--The white empty space in the middle-->
				<div class="mid"></div>
				
				<!--The container of the last section-->
				<div class="last">
				
					<!--The left box-->
					<div class="left">
					
						<!--The header of the left box-->
						<div class="left-head"></div>
					</div>
					
					<!--The right box-->
					<div class="right">
					
						<!--The header of the right box-->
						<div class="right-head"></div>
					</div>
				</div>		
			</div>
			
			<!--The footer text-->
			<div class="footer">CMSC2 Midyear 2015</div>
		</div>
	</body>

</html>
