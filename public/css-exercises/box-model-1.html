In HTML, each element is represented as a rectangular box. In CSS, each of these rectangular boxes is described using the standard box model. This model describes the content of the space taken by an element. It consists of: margin, border, padding, and the actual content.

The image below illustrates the box model:

Parts of the Box Model

Area Description
content the content of the box, where text and images appear
padding clears an area around the content; transparent
border a border that goes around the padding and content
margin Clears an area outside the border; transparent

Example

Hooray! The Golden State Warriors won the NBA Finals!

<p> Hooray! The Golden State Warriors won the NBA Finals! </p>
p{
	background-color: lightblue;
	width: 200px;
	/*The box model properties*/
	padding: 25px;
	border: 25px solid navy;
	margin: 25px;
}

Padding

The CSS padding properties define the space between the element border and the element content. The padding is affected by the background color of the element. The padding can be controlled using the padding-top, padding-bottom, padding-left, padding-right properties or the shorthand padding CSS property.

Example

Hooray! The Golden State Warriors won the NBA Finals!

p{
	background-color: lightblue;
	width: 200px;
	
	padding-top: 25px;
	padding-right: 25px;
	padding-bottom: 25px;
	padding-left: 25px;
	/*SAME AS*/
	padding: 25px;
}

You can specify different padding for each side.

Example

Hooray! The Golden State Warriors won the NBA Finals!

p{
	background-color: lightblue;
	width: 200px;
	
	padding-top: 10px;
	padding-right: 20px;
	padding-bottom: 30px;
	padding-left: 40px;
	/*SAME AS*/
	padding: 10px 20px 30px 40px;
	/*first value for top, second for right, third for bottom and fourth for left*/
}

Padding - shorthand property
The shorthand padding property can have from one to four values.
  • padding: 25px 50px 75px 100px;
    • top padding is 25px
    • right padding is 50px
    • bottom padding is 75px
    • left padding is 100px

  • padding: 25px 50px 75px;
    • top padding is 25px
    • right and left paddings are 50px
    • bottom padding is 75px

  • padding: 25px 50px;
    • top and bottom paddings are 25px
    • right and left paddings are 50px

  • padding: 25px;
    • all four paddings are 25px

Border

The CSS border properties allow you to specify the style, size, and color of an element's border.

Border-style
The border-style property specifies what kind of border to display.
Note: None of the border properties will have ANY effect unless the border-style property is set.

Different border-style property values:

Example

none; Defines no border

dotted; Defines a dotted border

dashed; Defines a dashed border

solid; Defines a solid border

double; Defines two borders. The width of the two borders are the same as the border-width value

groove; Defines a 3D grooved border. The effect depends on the border-color value

ridge; Defines a 3D ridged border. The effect depends on the border-color value

inset; Defines a 3D inset border. The effect depends on the border-color value

outset; Defines a 3D outset border. The effect depends on the border-color value


Border-width
The width is set in pixels, or by using one of the three pre-defined values: thin, medium, or thick.
Note: The "border-width" property does not work if it is used alone. Use the "border-style" property to set the borders first.

Example

Hooray! The Golden State Warriors won the NBA Finals!

p{
	background-color: lightblue;
	width: 200px;
	padding: 25px;
	
	/*Don't forget to set the border-style property first.*/
	border-style:solid;
	border-width:5px;
}

Example

Hooray! The Golden State Warriors won the NBA Finals!

p{
	background-color: lightblue;
	width: 200px;
	padding: 25px;
	
	/*Don't forget to set the border-style property first.*/
	border-style:solid;
	border-width:25px;
}
Notice that the color of the border is black since we did not set it yet.


Border-color
The border-color property is used to set the color of the border. The color can be set by:
  • name - specify a color name, like red
  • RGB - specify a RGB value, like rgb(255,0,0)
  • Hex - specify a hex value, like #ff0000
If the border color is not set it is inherited from the color property of the element.
Note: The "border-color" property does not work if it is used alone. Use the "border-style" property to set the borders first.

Example

Hooray! The Golden State Warriors won the NBA Finals!

p{
	background-color: lightblue;
	width: 200px;
	padding: 25px;
	
	/*Don't forget to set the border-style property first.*/
	border-style:solid;
	border-color:gray;
}

Example

Hooray! The Golden State Warriors won the NBA Finals!

p{
	background-color: lightblue;
	width: 200px;
	padding: 25px;
	
	/*All three border properties together*/
	border-style:solid;
	border-width:25px;
	border-color:gray;
}

Border - shorthand property
To shorten the code, it is also possible to specify all the individual border properties in one property. This is called a shorthand property.
The border property is a shorthand for the following individual border properties:
  • border-width
  • border-style (required)
  • border-color

Example

Hooray! The Golden State Warriors won the NBA Finals!

p{
	background-color: lightblue;
	width: 200px;
	padding: 25px;
	
	/*All three border properties together*/
	border-style:solid;
	border-width:25px;
	border-color:gray;
	/*EQUIVALENT TO*/
	border: 25px solid gray;
}

Margin

The margin clears an area around an element (outside the border). The margin does not have a background color, and is completely transparent.
The top, right, bottom, and left margin can be changed independently using separate properties. A shorthand margin property can also be used, to change all margins at once.

Example

Hooray! The Golden State Warriors won the NBA Finals!

p{
	background-color: lightblue;
	width: 200px;
	padding: 25px;
	border: 25px;
	
	margin-top: 25px;
	margin-right: 25px;
	margin-bottom: 25px;
	margin-left: 25px;
	/*SAME AS*/
	margin: 25px;
}

You can also specify different margins for the different sides of an element.

Example

Hooray! The Golden State Warriors won the NBA Finals!

p{
	background-color: lightblue;
	width: 200px;
	
	margin-top: 10px;
	margin-right: 20px;
	margin-bottom: 30px;
	margin-left: 40px;
	/*SAME AS*/
	margin: 10px 20px 30px 40px;
	/*first value for top, second for right, third for bottom and fourth for left*/
}

Margin - shorthand property
The shorthand margin property can have from one to four values.
  • margin: 25px 50px 75px 100px;
    • top margin is 25px
    • right margin is 50px
    • bottom margin is 75px
    • left margin is 100px

  • margin: 25px 50px 75px;
    • top margin is 25px
    • right and left margins are 50px
    • bottom margin is 75px

  • margin: 25px 50px;
    • top and bottom margins are 25px
    • right and left margins are 50px

  • margin: 25px;
    • all four margins are 25px

Margin: auto

This margin value is used for centering of blocks.

Example

Yay! I am at the center. No need for the deprecated <center></center> tag!
Oops! Can't reach the center. My margin wasn't auto. 
<div class="cent">
	Yay! I am at the center. No need for the deprecated <center></center> tag!
</div>
<div class="uncent">
	Oops! Can't reach the center. My margin wasn't auto.
