For this exercise, create an html file in the public/javascript-exercises folder named storing-information-2.html. Save your external script (js file) in public/script folder as storing-information-2.js. With the given HTML content and partial javascript file, modify the web page in such a way it will look exactly like the screenshot below. Make use of what you've learned about from the previous tutorial to be able to correctly imitate the web page.
Note: You cannot modify the html, only the javascript EXCEPT for adding the link to the external js.
